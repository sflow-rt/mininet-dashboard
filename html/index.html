Simple sFlow-RT real-time Mininet dashboard using Mininet flow analytics.

Mininet dashboard and Mininet weathermap provide additional examples.

Launch Mininet using the sflow.py helper script:

sudo mn --custom sflow-rt/extras/sflow.py --link tc,bw=10 --topo tree,depth=2,fanout=2

Click on peaks in the trend charts to see traffic at that time.
